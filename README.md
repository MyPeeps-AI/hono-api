# Simple TypeScript API with Hono framework

## Demo

### Chat endpoint `/chat`

- requires request parameter **message**
- talk to fictional character Sherlock Holmes

```bash
GET https://hono-api.mr-yenza.workers.dev/chat?message=hi%20sherlock
```

## Local Development

```bash
    > npm start
```

## Deployment

```bash
    > npm deploy
```
